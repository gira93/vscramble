# Vscramble

## Description
3x3 cube scramble made with VueJS 3, it can also be used as a PWA.
Made it just to have fun with VueJS 3.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
